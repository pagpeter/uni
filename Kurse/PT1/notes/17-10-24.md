
## Fibonacci

```c
#include <stdio.h>

int main(void) {
	int previous = 1;
	int preprevious = 0;
	int counter = 1;
	int upper = 20;
	
    // Handle special cases
    printf("%d: 0\n", counter++);
	printf("%d: 1\n", counter++);
	
	for (int i = 0; i < upper-2; i++) {
		int current = previous + preprevious;
		printf("%d: %d\n", counter++, current); 
		preprevious = previous;
		previous = current;
	}
}
```
